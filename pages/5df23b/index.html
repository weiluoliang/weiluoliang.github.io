docker安装MySQL

# 查看MySQL的版本

地址 ： https://hub.docker.com/_/mysql/tags (opens new window)

# docker安装MySQL

# 拉取MySQL 最新版本
docker pull mysql:latest

# 安装 并指定密码
docker run -itd --name mysql -p 3306:3306 -e MYSQL_ROOT_PASSWORD=123456 mysql

# 进入容器
