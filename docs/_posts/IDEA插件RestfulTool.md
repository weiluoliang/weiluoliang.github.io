### RestfulTool

1. 简介：

一套 Restful 服务开发辅助工具集
提供了一个 Services tree 的显示窗口
双击 URL 直接跳转到对应的方法定义
一个简单的 http 请求工具
支持 Spring 体系 (Spring MVC / Spring Boot)
支持 JAX-RS
支持 `Navigate -> Request Service` 搜索 Mapping (Ctrl + Alt + /)

2. 演示
  查找一个接口： **/user/list**    对应的 **controller**  ,  输入  `user`  关键词搜索
